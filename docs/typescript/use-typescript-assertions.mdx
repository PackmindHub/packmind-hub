# Use typescript assertion to check unknown types

<Severity severity={SeverityLevels.CRITICAL} />

---

<Categories categories={["typescript", "typesafe"]} />

### What is it?

Use TypeScript assertions to explicitly check unknown types, typically recognized by the usage of variables declared with 'unknown' type without subsequent type assertions to narrow the type before access or operations.
### Why apply it?

Using TypeScript assertions to check unknown types ensures type safety, preventing runtime errors and enhancing code readability by making type assumptions clear and explicit.
### How to Fix it?

Introduce type assertions or guards (such as 'typeof', 'instanceof', or custom type predicates) to safely narrow the 'unknown' type to a more specific type before any operations or access.
Read more:

https://www.typescriptlang.org/docs/handbook/basic-types.html#unknown
