# Use StringBuilder for concatenation of strings in loops

<Severity severity={SeverityLevels.MEDIUM} />

---

<Categories categories={["Java","Clean code","csharp"]} />

Avoid using the + operator to concatenate strings in loops.
StringBuilder implies that we are building a string.
That improves the code readability.

## Examples

### Example 1:

<!-- #example -->

#### Positive

<!-- #example_positive_description -->
Correct implementation following the practice.

<!-- #example_positive_code -->
```java
package org.main;

import java.util.ArrayList;
import java.util.List;

public class Main {

	public static void main(String[] args) {
		List<String> items = new ArrayList();
		
		StringBuilder sb = new StringBuilder();
		for (String item : items) {
		    sb.append(item);
		}
		String result = sb.toString();

	}

```
